Cara Membuat Web Statis Dengan Hugo

Aug 24, 2017 00:40 · 504 words · 3 minutes read

Untuk post pertama saya, disini saya akan membagikan tutorial dimana saya membuat web statis ini menggunakan hosting gratis dari github. Sebelumnya buat yang belum tau apa itu Hugo, Hugo is a fast and modern static site generator written in Go, and designed to make website creation fun again. Intinya ini salah satu Generator Web Statis yang ringan dibanding dengan SSG yang lain dan menggunakan bahasa GO. Kita dapat menulis artikel secara offline menggunakan text editor favorit kita, jadi kita tidak perlu susah payah untuk mem-preview hasil dari artikel kita dan dapat kita publish disaat kita sudah online nantinya. Oke tanpa basa basi langsung saja Let’s do that !

1. Instalisasi Hugo dan GO

Sebelum mengikuti tutorial ini lebih baiknya kalian sudah menginstall GIT pada OS kalian. Saya disini menggunakan versi Hugo yang terakhir release saat artikel ini ditulis yaitu v.026. Kalian bisa download Hugo sesuai dengan versi OS kalian, disini saya menggunakan linux untuk membuat websitenya. Jadi untuk OS lain seperti Windows/Mac bisa menyesuaikan :D. Berikut link hugo nya:

https://github.com/gohugoio/hugo/releases

Setelah kalian download install seperti biasa menggunakan dpkg di terminal.

sudo dpkg -i hugo_0.26_Linux-64bit.deb

Jika sudah selesai proses install, silahkan cek version hugonya.

hugo version

Kemudian kita install bahasa GO pada komputer kita.

sudo apt install go

Periksa versi go.

go version

2. Membuat Website Baru

Setelah kita berhasil instalisasi Hugo dan GO selanjutnya kita akan membuat websitenya. Sangat mudah membuat website baru di Hugo menggunakan terminal linux kalian.

Untuk membuat website baru di Hugo, menggunakan perintah ini :

hugo new site [nama web]

Otomatis hugo akan membuat folder baru sesuai nama web nya

Lalu masuk ke dalam folder

cd [nama folder]

Dan kita sudah berhasil membuat web yang diinginkan

3. Memilih Tema dan Menginstallnya

Cukup mudah bukan langkah-langkah membuat website nya sampai nomor 3 ini. Oke di langkah ketiga ini kita akan menginstall tema pada Hugo, sangat mudah sekali.

Pertama kita pilih dahulu tema hugo yang ingin kita install pada web kita. Untuk download nya bisa melalui website ini:

https://themes.gohugo.io

Disini saya akan install tema Hyde.

cd themes
git clone https://github.com/spf13/hyde.git

Kemudian kita kembali ke folder utama, dan edit file config.toml

cd ..
nano config.toml

Lalu tambahkan sebagai berikut:

languageCode = "en-us"
title = "Blog | HBS"
#tambahkan seperti dibawah ini sesuai nama tema kalian
theme = "hyde"

Lalu kita sinkronkan ke hugo :

hugo -t [nama tema]

Setelah selesai, kita bisa melihat bagaimana bentukan website kita menggunakan localhost

hugo server

4. Cara Menulis Artikel di Hugo

Setelah website kita jadi, kita bisa membuat artikel menggunakan text editor kesayangan kita. Ohh iya, di Hugo kita menulis artikel menggunakan bahasa Markdown (.MD) Kalian bisa melihat list commmand pengganti syntax HTML disini. Tanpa basa basi lagi langsung saja kita membuat artikel nya:

hugo new post/[nama-artikel].md

Contoh :

hugo new post/belajar-membuat-artikel.md

Filenya berada di /contents/post/. Jangan lupa mengubah nilai draft dari true menjadi false agar bisa tampil pada website.

Yeayy!!! Kita udah berhasil membuat web statis menggunakan Hugo.
